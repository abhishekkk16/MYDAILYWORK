# ATM Interface – Java OOP Project

This project is a console-based ATM simulation developed using Core Java.  
It demonstrates Object-Oriented Programming principles and basic banking operations.

# Features

- PIN Authentication
- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-driven Interface

# Technologies Used

- Java (Core Java)
- OOP Concepts
- IntelliJ IDEA
- Maven Project Structure

# Project Structure

ATMInterface
│── BankAccount.java
│── ATM.java
│── Main.java

# OOP Concepts Implemented

- Encapsulation (Private variables with controlled access)
- Constructor usage
- Method abstraction
- Object interaction
- Separation of concerns

# How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`
4. Enter PIN to access ATM menu
